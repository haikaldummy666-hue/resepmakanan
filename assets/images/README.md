## Format Rekomendasi

- **Format File**: JPG, PNG (rekomendasi JPG untuk optimasi ukuran)
- **Ukuran Gambar**: 400 x 300 px atau rasio 4:3
- **Ukuran File**: Max 100KB per gambar

## Cara Menggunakan

Setiap thumbnail otomatis direferensikan di `data/recipes.ts` dengan path:
```
/assets/images/[recipe-id].jpg
```

Contoh: Recipe dengan ID `telur-ceplok-nasi` akan mencari file `telur-ceplok-nasi.jpg` di folder ini.
